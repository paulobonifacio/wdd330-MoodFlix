Welcome to MoodFlix

MoodFlix is a mood-based movie and TV show recommender built using:

  • Hugging Face Transformers API – for detecting user mood from text
  • TMDb API – for retrieving movies, images, genres, and ratings

Features

  • Mood-based suggestions
  • Trending and manual search
  • Favorites and local storage
  • Responsive, animated UI

Developer

This app was created as part of a frontend web development project.
